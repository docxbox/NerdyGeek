# Privacy Policy

NerdyGeek is an open-source documentation lookup tool for coding agents.

## What NerdyGeek Does

NerdyGeek accepts a documentation query and optional repository context such as
`package.json`, discovers relevant documentation sources, retrieves public
documentation pages, and returns a structured summary with source links.

## Data NerdyGeek May Process

Depending on how it is deployed, NerdyGeek may process:

- search queries submitted by a user or coding agent
- package metadata provided in tool input, such as dependency names and versions
- public documentation pages retrieved from third-party websites
- basic operational logs such as timestamps, errors, and cache metadata

## What NerdyGeek Does Not Intend to Collect

The project is not designed to collect or sell personal data. It is not built
to require account creation, advertising profiles, or marketing trackers.

## Third-Party Requests

When NerdyGeek retrieves documentation, it may send requests to third-party
documentation websites or search providers. Those services may receive network
metadata such as IP address, user agent, timestamps, and requested URLs in
accordance with their own privacy practices.

## Local and Self-Hosted Use

If you run NerdyGeek locally or self-host it, you control your deployment,
logs, retention, access controls, and any data handling choices. You are
responsible for configuring the service in a way that meets your own privacy
and compliance requirements.

## Data Retention

By default, the project uses a short-lived cache for documentation lookups.
Any longer-term logging, analytics, or storage depends on how an operator
deploys the service.
